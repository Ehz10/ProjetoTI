O Salto de Paraquedas Tandem - Skydive, no Algarve, é uma experiência única, sem igual!

É o salto de paraquedas (Salto Tandem) mais popular do mundo, perfeito para quem procura uma experiência única, cheia de emoções fortes, sem exigir experiência prévia.

Nunca irá estar sozinho, pois o paraquedas foi criado para duas pessoas. Neste caso, para o cliente e para o instrutor, profissional altamente qualificado e certificado pela FPPQ: Federação Portuguesa de Paraquedismo. E, apesar de esta ser uma actividade de aventura, para corajosos, está sempre em segurança!
